# Boilerplate for Meteor-Vue app
Good for an enterprise start or learning purpose.

## Highlights: 
- Back End: Meteor
- Front End: Vue
- Template Engine: Pug
- CSS Framework: Boostrap-Vue
- Vuex
- vue-supply for managing reactive and realtime data retrieve from Meteor's subscription.
- Vue Router
